About CmapusXplorer

CmapusXplorer is a web application that allows users to explore the campus of the University of Waterloo. Users can view the buildings on campus, the rooms in each building, and the courses that are offered in each room. Users can also search for buildings, rooms, and courses. The application is built using the Django framework and the Google Maps API.
